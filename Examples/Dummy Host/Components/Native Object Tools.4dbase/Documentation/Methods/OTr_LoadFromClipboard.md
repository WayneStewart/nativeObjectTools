Delegates all parsing to OTr\_LoadFromText, including BOM stripping and top-level array wrapping.Returns 0 if the clipboard contains no text or invalid JSON.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||Result|Integer|⬅️|New OTr handle,                         or 0 if loading failed|
