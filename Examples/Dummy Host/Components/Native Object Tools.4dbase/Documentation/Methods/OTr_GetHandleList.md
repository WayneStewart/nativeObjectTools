When an object is cleared with *OT Clear*, the object's handle is removed from the list.*OT GetHandleList* retrieves this internal list into an array. This is mainly of use in debugging. Normally you would have no need to use this method.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||outHandles|Pointer|⬅️|Pointer to Longint array to receive active handles (outHandles)|
