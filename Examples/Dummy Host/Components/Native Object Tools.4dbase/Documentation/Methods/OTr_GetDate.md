If the *FailOnNoItem*option is set, an error is generated and *OK* is set to zero.If an item with the given tag exists and has the type *Is Date*, the value of the requested item is returned.If an item with the given tag exists and has any other type, *OK* is set to zero, and a null date is returned.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path (inTag)||Result|Date|⬅️|Stored value, or empty date on missing/invalid|
