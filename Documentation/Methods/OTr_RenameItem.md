For example, renaming "foo.bar.old\_name" to "new\_name" gives access via "foo.bar.new\_name".BLOB and Picture references are moved, not copied (no new slot).Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Full tag path of the item to rename (inTag)||inNewTag|Text   |➡️|New leaf name (inNewTag)|
