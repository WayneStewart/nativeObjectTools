The array is serialised as a 4D Object with metadata properties (arrayType, numElements, currentItem) and one string-keyed property per element ("0" through "n"). Date and Time elements are stored as ISO text via OTr\_uDateToText and OTr\_uTimeToText.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path for the array (inTag)||inArray|Pointer|➡️|Pointer to the source 4D array (inArray)|
