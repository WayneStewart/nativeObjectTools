Existing files are overwritten without warning. Delegates all serialisation to OTr\_z\_XMLWriteObject.See OT SaveToXML for a description of the XML schema. Shadow-type keys (leafKey$type) are included or excluded according to OT IncludeShadowKey (default: True). Use OT SaveToXML to obtain the XML as a Text value instead.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inFilePath|Text   |➡️|Full path of the target file||inPrettyPrint|Boolean|➡️|True for indented output; default True (optional)|
