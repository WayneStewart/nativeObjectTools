Note: byte counts will not match the legacy plugin exactly.**WARNING: Changed Behaviour**The value is based on OTr's native 4D serialisation of the object, not the in-memory byte size of the legacy ObjectTools plugin structure. Treat values as OTr-only relative estimates.**ORIGINAL DOCUMENTATION***OT ObjectSize* returns the total size of an object in memory. If *inObject* is not a valid object handle, an error is generated, *OK* is set to zero, and zero is returned.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||Result|Integer|⬅️|Approximate size in characters, or 0 on error|
