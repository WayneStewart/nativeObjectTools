If the *FailOnNoItem* option is set, an error is generated and *OK* is set to zero.If an item with the given tag exists and has the type *Is BLOB*, the value of the requested item is returned.If an item with the given tag exists and has any other type, *OK* is set to zero and an empty BLOB is returned.Warning: Because of the problems related to *OT GetBLOB*, it is recommended that you use this command instead.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path (inTag)||Result|Blob|⬅️|Stored BLOB, or empty BLOB on failure|
