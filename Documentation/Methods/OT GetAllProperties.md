Delegates to OT GetAllNamedProperties with an empty tag. Item names are returned in indeterminate order.**ORIGINAL DOCUMENTATION***OT IsEmbedded* tests the item referenced by *inTag* to see if it is an embedded object.If *inObject* is not a valid object handle or if no item in object has the given tag, an error is generated, *OK* is set to zero, and zero is returned.If an item with the given tag exists and has the type *OT Is Object*, 1 is returned.If an item with the given tag exists and has any other type, zero is returned.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|вЮ°пЄП|OTr inObject||outNames|Pointer|вђЕпЄП|Receives item names (Text array)||outTypes|Pointer|вђЕпЄП|Receives OT type constants||outItemSizes|Pointer|вђЕпЄП|Receives item sizes (Longint array)||outDataSizes|Pointer|вђЕпЄП|Receives data sizes (Longint array)|
