Embedded objects are deleted recursively. BLOB and Picture parallel array slots referenced by the deleted item (or any sub-items) are released.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag of the item to delete (inTag)|
