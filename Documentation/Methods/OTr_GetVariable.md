Scalar variable values are stored as "var:typeName:value".Array variable values are stored as sub-objects; retrieval is delegated to OTr\_GetArray.**ORIGINAL DOCUMENTATION***OT GetVariable* retrieves a value from *inObject* at *inTag* into the variable pointed to by *varPtr*.Every 4D variable type except 2D arrays can be retrieved with this command.If *inObject* is not a valid object handle, an error is generated and OK is set to zero.If no item in the object has the given tag, nothing happens.If an item with the given tag exists and the stored type matches the type of the destination variable, the variable's data is replaced with the stored data.If the stored type does not match the type of the destination variable, an error is generated and OK is set to zero.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path (inTag)||outVarPointer|Pointer|➡️|Pointer to variable to receive the value|
