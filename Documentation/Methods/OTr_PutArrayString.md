OT *SizeOfArray* ( *inObject; inTag* )), the value of the requested element is set to *inValue*.If an item with the given tag exists and has any other type, or if the index is out of range, an error is generated and *OK* is set to zero.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path to the array item (inTag)||inIndex|Integer|➡️|Element index, 1-based; 0 = default element (inIndex)||inValue|Text   |➡️|Value to store (inValue)|
