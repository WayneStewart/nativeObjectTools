The resulting file is fully interchangeable with files written by OTr\_SaveToXMLFile.Use OTr\_SaveToXMLSAX to obtain the XML as a Text value instead.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr handle||inFilePath|Text   |➡️|Full path of the target file||inPrettyPrint|Boolean|➡️|True for indented output; default True (optional)|
