The target array must already be declared with an appropriate type. Elements are converted from stored text/values via the OTr\_u utility methods as required.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path to the array item (inTag)||outArray|Pointer|➡️|Pointer to the destination 4D array (outArray)|
