If growing, new elements are appended with the array type's default value. If shrinking, excess elements are removed from the end.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path to the array item (inTag)||inSize|Integer|➡️|New number of elements (inSize)|
