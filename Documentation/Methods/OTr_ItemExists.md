Returns 1 if found, 0 if not. An invalid handle generates an error; a missing path does not (this is a query, not a mutation).Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag of the item to query (inTag)||Result|Integer|⬅️|1 if item exists, 0 if not|
