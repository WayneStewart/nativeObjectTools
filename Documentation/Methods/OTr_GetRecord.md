Field matching is by name; unrecognised properties are silently ignored and unmatched fields are untouched.Date and time fields are restored from text via OTr\_uTextToDate and OTr\_uTextToTime. Picture and BLOB fields are decoded from base64.Adapted from OBJ\_ToRecord by Cannon Smith.**ORIGINAL DOCUMENTATION***OTr\_GetRecord* writes field values from the snapshot stored at *inTag* into the current record of *inTable*.It is up to the caller to ensure a record is loaded in read/write mode before calling this method, and to save the record afterwards.If *inObject* is not a valid object handle, an error is generated and OK is set to zero.If *inTable* is not a valid table number, an error is generated and OK is set to zero.If no item in the object has the given inTag, or the item is not a record snapshot sub-object, an error is generated and OK is set to zero.Note: Unlike the legacy OTr\_GetRecord, this method writes snapshot values into whatever record is currently loaded.No database read is performed.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path (inTag)||inTable|Integer|➡️|Table number whose current record to populate (inTable)|
