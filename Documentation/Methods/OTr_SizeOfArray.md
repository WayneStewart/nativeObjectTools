Does not count element 0.Returns 0 if the handle is invalid, the tag does not exist, or the tag does not reference an array item.**ORIGINAL DOCUMENTATION***OT SizeOfArray* returns the number of elements in an array item within an object.If *inObject* is not a valid object handle, if no item in the object has the given tag, or if the item’s type is not an array type, an error is generated, *OK* is set to zero, and zero is returned.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path to the array item (inTag)||Result|Integer|⬅️|Number of elements (not counting element 0), or 0|
