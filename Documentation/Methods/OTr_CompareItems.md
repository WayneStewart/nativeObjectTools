Both handles may be the same object.Returns 1 if items are identical, 0 if not, -1 on error.Objects are compared via JSON serialisation. Collections are compared element-by-element via JSON serialisation. BLOBs are compared by content. Text (including encoded dates, times, pointers) is compared with exact string equality.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inSourceObject|Integer|➡️|OTr inSourceObject||inSourceTag|Text   |➡️|Source item tag (inSourceTag)||inCompareObject|Integer|➡️|OTr inCompareObject||inCompareTag|Text   |➡️|Comparison item tag (inCompareTag)||Result|Integer|⬅️|1 if identical, 0 if not, -1 on error Wayne Stewart, 2026-04-01 - Uses helper equality methods for text, object,   BLOB, and Picture comparisons.
