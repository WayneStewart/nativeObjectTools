Elements are filled using the stored arrayType and string-indexed keys "0" through "numElements". Type conversions are applied via the OTr\_u\_ utility methods for Date, Time, Pointer, BLOB, and Picture types. Called internally by OTr\_GetArray and typed element Get methods. Callers are responsible for locking.Attributes: Private, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||arrayObj|Object |➡️|OTr array object||arrayPtr  |Pointer|➡️|Pointer to target 4D array|
