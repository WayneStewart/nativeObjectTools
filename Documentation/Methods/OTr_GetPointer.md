The stored text is deserialised via OTr\_uTextToPointer. Returns a Null pointer on any error or missing tag.**ORIGINAL DOCUMENTATION***OT GetPointer* retrieves a Pointer value from *inObject*.If *inObject* is not a valid object handle, an error is generated, OK is set to zero, and a Null pointer is returned.If no item in the object has the given inTag, a Null pointer is returned. If the FailOnItemNotFound option is set, an error is generated and OK is set to zero.If an item with the given inTag exists and has the type *Is Pointer*, the value of the requested element is returned.If an item with the given inTag exists and has any other type, an error is generated, OK is set to zero, and a Null pointer is returned.Warning: Under no circumstances should you attempt to store a pointer to a local or process variable in a compiled database and then try to retrieve that pointer in another process.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path (inTag)||outPointer|Pointer|➡️|Receives the retrieved pointer (outPointer)|
