Variable addresses differ between processes in compiled mode.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path (inTag)||inValue|Pointer|➡️|Pointer to store (inValue)|
