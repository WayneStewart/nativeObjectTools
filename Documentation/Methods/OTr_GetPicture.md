Returns an empty Picture on any error or missing tag.**ORIGINAL DOCUMENTATION***OTr\_GetPicture* retrieves a Picture value from *inObject*.If *inObject* is not a valid object handle, an error is generated, OK is set to zero, and an empty Picture is returned.If no item in the object has the given inTag, an empty Picture is returned. If the FailOnItemNotFound option is set, an error is generated and OK is set to zero.If an item with the given inTag exists and has the type *Is Picture*, the value of the requested item is returned.If an item with the given inTag exists and has any other type, an error is generated, OK is set to zero, and an empty Picture is returned.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path (inTag)||Result|Picture|⬅️|Stored Picture, or empty Picture on failure|
