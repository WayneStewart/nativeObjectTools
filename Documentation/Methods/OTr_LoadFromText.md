<!--OTr_LoadFromText (inJSON) -> LongintinJSON : Text : ➡️ : JSON string to parseResult : Integer : ⬅️ : New OTr handle, \                         or 0 if parsing failed-->## OTr\_LoadFromTextOTr\_LoadFromText (inJSON) --> LongintParses a JSON string and stores the resulting object in a new OTr handle. If the top-level JSON value is an array, it is automatically wrapped in an object keyed "OBJ" to preserve OTr's object-at-root convention. JSON parse errors are caught silently; check the return value to know whether parsing succeeded.Returns 0 if inJSON is empty or cannot be parsed.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inJSON|Text|➡️|JSON string to parse||Result|Integer|⬅️|New OTr handle, \                         or 0 if parsing failed|
