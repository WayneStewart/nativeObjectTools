The format is 4D-internal and is not portable outside 4D. Use OT SaveToGZIP when portability across systems is required.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||Result|Blob|⬅️|Compressed 4D-native blob, \                          or empty blob if the handle is invalid|
