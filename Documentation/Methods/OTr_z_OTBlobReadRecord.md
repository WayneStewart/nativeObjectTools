Field names are not present in the BLOB, so stable fieldN properties are always written. When the source table number is valid in the current database, matching field-name properties are also written so OTr\_GetRecord can restore into a current record. Attributes: Private, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inBlob|Blob   |➡️|Legacy ObjectTools object BLOB||ioOffset|Pointer|↔️|Offset at record payload start; receives end offset||Result|Object|⬅️|Record snapshot object, or Null on failure //|
