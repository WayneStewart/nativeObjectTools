BLOB, Picture, and Pointer are not sortable. Returns False and reports an error for any failure; OTr\_z\_ResolvePath reports its own errors.Attributes: Private, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||params|Object|➡️|see above||Result||⬅️|  Boolean : True if the pair is valid|
