Used to verify type compatibility before performing array operations.Returns -1 if the handle is invalid, the tag does not exist, or the tag does not reference an array.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path to the array item (inTag)||Result|Integer|⬅️|Stored array type constant, or -1|
