Files written here are intended for OTr\_LoadFromFile and are not compatible with legacy ObjectTools.Note: TEXT TO DOCUMENT with "UTF-8" writes a BOM at the start of the file. This is harmless for inspection purposes; if you later parse the file with JSON Parse, 4D handles the BOM correctly for locally written files.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inFilePath|Text   |➡️|Full path of the target file (inFilePath)||inPrettyPrint|Boolean|➡️|True for indented output; default True (optional)|
