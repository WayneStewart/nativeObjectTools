Fnd\_Art) (c) "Specific method name" - write comments for that method The second parameter - True: generate a tooltip The third parameter - True: Exclude Private methods The fourth parameter - True: Don't show a final "Finished" alert The fifth parameter  - True: Run inline in the calling process (no New process spawn) Use in headless/CI contexts where process management is undesirable.Attributes: Private, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||methodName|Text|➡️|Method Name, Prefix of empty (all methods)||toolTip|Boolean|➡️|Write in the summary section (for the Tooltip)||excludePrivate|Boolean|➡️|Exclude Private methods||silent|Boolean|➡️|Show or Don't show a Finished alert after task complete||inline|Boolean|➡️|Run inline in calling process rather than spawning a new process|
