Reads the list of methods in Resources/OTr\_OTBlockMethods.json, gets each method's source through METHOD GET CODE, and comments or uncomments each standardized ObjectTools plugin block by adding or removing wrapper lines next to the delimiters. The target state is selected from OTr\_z\_PluginShouldWork. If the plugin should work, OT blocks are uncommented. If the plugin should not work, OT blocks are commented. Attributes: Private, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||suppressAlert|Boolean|➡️|Optional. True to suppress final ALERT.|
