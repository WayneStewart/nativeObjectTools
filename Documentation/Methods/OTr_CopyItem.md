The destination need not exist; it will be created. Source and destination handles may be the same. Embedded objects are deep-copied via OB Copy.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inSourceObject|Integer|➡️|OTr inSourceObject||inSourceTag|Text   |➡️|Source item tag (inSourceTag)||inDestObject|Integer|➡️|OTr inDestObject||inDestTag|Text   |➡️|Destination item tag (inDestTag)|
