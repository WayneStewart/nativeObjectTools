The handler method must accept four parameters:- *message* (Text): a description of the error- *method* (Text): the name of the ObjectTools method that was called- *object* (LongInt): the object handle involved, or zero if none- *tag* (Text): the tag being referenced, or empty if noneWhether or not an error handler is set, *OK* is set to zero whenever an error occurs.*OT SetErrorHandler* returns the old handler so that you may dynamically change error handling within your code.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inNewHandler|Text|➡️|Method name to use as error handler (inNewHandler)||Result|Text|⬅️|Previously registered handler method name|
