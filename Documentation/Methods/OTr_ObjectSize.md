All scalar, text, date, boolean, and nested object properties are included in the JSON output. Native BLOB and Picture properties are not emitted by JSON Stringify and are therefore not counted; this is consistent with the legacy plugin's approximation contract. No loop over individual keys is required because all content — including embedded objects at any depth — is fully serialised by a single JSON Stringify call.Byte counts will not match the legacy plugin.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||Result|Integer|⬅️|Approximate size in characters, or 0 on error|
