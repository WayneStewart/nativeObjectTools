This is an approximation; byte counts will not match the legacy plugin.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|вћЎпёЏ|OTr inObject||Result|Integer|в¬…пёЏ|Approximate size in bytes, or 0 on error|
