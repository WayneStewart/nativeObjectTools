Note: byte counts will not match the legacy plugin exactly.**ORIGINAL DOCUMENTATION***OT ObjectSize* returns the total size of an object in memory. If *inObject* is not a valid object handle, an error is generated, *OK* is set to zero, and zero is returned.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||Result|Integer|⬅️|Approximate size in characters, or 0 on error|
