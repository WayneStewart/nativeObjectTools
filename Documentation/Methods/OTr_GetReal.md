If the *FailOnNoItem* option is set, an error is generated and *OK* is set to zero.If an item with the given tag exists and has the type *Is Real*, the value of the requested item is returned.If an item with the given tag exists and has any other type, *OK* is set to zero, and zero is returned.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path (inTag)||Result|Real|⬅️|Stored value, or 0 on missing/invalid|
