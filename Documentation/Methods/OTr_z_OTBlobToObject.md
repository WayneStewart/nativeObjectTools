Also supports the compact Guy sample layout with ASCII keys, marker 2 text, and marker 18 text arrays. Attributes: Private, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inBlob|Blob|➡️|Legacy ObjectTools object BLOB||Result|Object|⬅️|Native OTr object storage shape, or Null on failure //|
