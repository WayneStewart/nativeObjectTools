The blob is expanded automatically if it is compressed before reading via BLOB TO VARIABLE.Returns 0 if the blob is empty or cannot be decoded.**NOTE:** there is no equivalent Object Tools commandAttributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inBlob|Blob|➡️|Compressed 4D-native blob \||Result|Integer|⬅️|New OTr handle, \                         or 0 if loading failed|
