Pass False for compact single-line output. Use OTr\_SaveToXMLFile to write directly to a file.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inPrettyPrint|Boolean|➡️|True for indented output; default True (optional)||Result|Text|⬅️|XML string, or empty text if handle is invalid|
