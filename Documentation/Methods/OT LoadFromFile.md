The file must have been written as a plain UTF-8 JSON text file (e.g. by OT SaveToFile).**NOTE:** there is no equivalent Object Tools commandAttributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inFilePath|Text|➡️|Full path of the source file||Result|Integer|⬅️|New OTr handle,                         or 0 if loading failed|
