Internal \_\_otr\_ properties are excluded.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag of an embedded object (optional) (inTag)||Result|Integer|⬅️|Item count (excluding internal __otr_ properties), or 0 Wayne Stewart, 2026-04-01 - Updated OB Keys usage for collection return.|
