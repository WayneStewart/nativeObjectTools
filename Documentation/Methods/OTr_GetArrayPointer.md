A pointer stored from one process may not resolve correctly in another.OK is unchanged on success; set to 0 on any failure.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path to the array item (inTag)||inIndex|Integer|➡️|Element index, 1-based; 0 = default element (inIndex)||Result|Pointer|⬅️|Element value, or Null on any failure|
