The blob is expanded and decoded as UTF-8 text before being parsed as JSON. Delegates parsing to OTr\_LoadFromText.Returns 0 if the blob is empty or cannot be decoded.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inBlob|Blob|➡️|GZIPed JSON blob \||Result|Integer|⬅️|New OTr handle, \                         or 0 if loading failed|
