Returns empty text if the BLOB is empty. No sentinel prefix is produced.Attributes: Private, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||theBlob|Blob|➡️|The BLOB to encode||Result|Text|⬅️|Base64-encoded representation|
