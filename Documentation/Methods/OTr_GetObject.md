If this routine successfully returns a new object, the new object handle is added to the ObjectTools handle list.If the object is not a valid object handle, an error is generated, *OK* is set to zero, and a null object handle (0) is returned.If no item in the object has the given tag, a null object handle is returned.If an item with the given tag exists and has the type *OT Is Object (114)*, its contents are returned as a new object.If an item with the given tag exists and has any other type, an error is generated, *OK* is set to zero, and a null object handle is returned.Warning: This method creates and returns a new object in memory. You are responsible for clearing it when you no longer need it by calling *OT IsObject*.Attributes: Shared, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||inObject|Integer|➡️|OTr inObject||inTag|Text   |➡️|Tag path to embedded object (inTag)||Result|Integer|⬅️|New handle containing a copy of the embedded object, or 0|
