<!--OTr_zIsShadowKey ($key_t : Text) -> Booleankey : Text : ➡️ : Property key to testResult : Boolean : ⬅️ : True if $key_t is a shadow key-->## OTr\_zIsShadowKeyOTr\_zIsShadowKey ($key\_t : Text) --> BooleanReturns True if $key\_t is an OTr shadow-type key (a key ending in the reserved "$type" suffix used to carry the original OT type constant for text-collapsed scalars). Used by the property enumerators (OTr\_GetNamedProperties, OTr\_GetAllNamedProperties, OTr\_GetItemProperties) to hide shadow keys from callers, and by OTr\_zMapType to avoid recursively mapping a shadow as if it were a user property. Paired with OTr\_zShadowKey which constructs shadow keys.Attributes: Private, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||key|Text|➡️|Property key to test||Result|Boolean|⬅️|True if $key_t is a shadow key|
