If $ref\_t begins with "blob:" the corresponding slot in <>OTR\_Blobs\_ablob is cleared and marked free. If it begins with "pic:" the corresponding slot in <>OTR\_Pictures\_apic is marked free. Any other value is silently ignored. Must be called within a held registry lock.Attributes: Private, Invisible, Preemptive indifferent| Parameter | Type |  | Description || --- | --- | --- | --- ||ref|Text|➡️|Stored element value to inspect|
